Collaboration Research with BRIN, Telkom University, and Kanazawa University, 2026
Computer Vision Medical Imaging Deep Learning

Overview

ASU (Annotation Segmentation Utility) is a desktop GUI application that integrates Meta AI's Segment Anything Model (SAM) for semi-automatic image annotation and segmentation. The tool is designed to accelerate and standardize ground-truth generation for deep learning model training, particularly in medical and scientific imaging workflows.

The application has been demonstrated on MRI brain tumor segmentation, where it significantly reduced annotation time compared to fully manual methods, while maintaining high boundary accuracy.

Key Features

  • SAM model integration with selectable variants (ViT-H and ViT-B)
  • Interactive click-based prompting (positive / negative points)
  • Project-based batch image management (New / Open / Save Project)
  • Multi-class labeling with customizable class names
  • Export annotations in YOLO format for downstream model training
  • CPU / GPU device selection with automatic fallback to CPU
  • Zoom and fit-to-window controls for detailed annotation
  • Undo / Redo and Delete Annotation controls

Tech Stack

Python SAM (Meta AI) SAM-2 PyTorch OpenCV PyQt5 NumPy YOLO Export

Application

This tool bridges the gap between foundation models and practical annotation workflows. It is particularly relevant for:

  • Generating training data for organ segmentation in radiotherapy planning
  • Semi-automatic delineation in low-contrast medical images
  • Building labeled datasets for surgical scene understanding
  • Rapid prototyping of segmentation pipelines in research environments
